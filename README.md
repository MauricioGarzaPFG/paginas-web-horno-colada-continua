# Páginas web - Sistema de monitoreo y control
Interfaz web desarrollada en HTML para el monitoreo y control 
de un horno de colada continua mediante PLC S7-1200.

## Archivos para páginas de interfaz de monitoreo y control
- index.html — Página principal y navegación
- temperatura_corriente.html — Monitoreo de corriente y temperatura
- sistema_agua.html — Monitoreo del sistema de agua
- control_onoff.html — Control de encendido y apagado
- IO.html — direcciones del PLC para lectura y escritura
- On.html y Off.html  — Archivos para enviar señales de encendido y apagado

## Archivos para pruebas de servidor
- pruebaWebserver.html — Interfaz para realizar la prueba
- IOContador.html — direcciones del PLC para lectura y escritura
- Suma.html — Archivo para enviar señal al contador
- Reinicio.html — Archivo para reiniciar contador
